# Variables
## Variable Types
- global Variable
    by default all variables are considered global
    a = "hello"
- local variables
    functional and block scope
    local a = "hello"
- table field
    special type of variable which can hold anything except nil
    similer to JS object
    local a = { a = 1, b = "two" }


    multiple assignments
    a,b = 3,5
