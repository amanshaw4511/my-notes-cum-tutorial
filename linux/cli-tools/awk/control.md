# Control Flow

## If
    if (<cond>) {
        <statement>
    }

## If else
    if (<cond>) {
        <statement>
    } else {
        <statement>
    }

## For loop
    for (i = 1; i<=10; i++) {
        <statement>
    }

## While loop
    while (<cond>) {
        <statement>
    }

## Do While loop
    do {
        <statement>
    } while (<cond>)



## Break, Continue and Exit
    break           => stop the loop
    continue        => continue to next iteration
    exit            => stop execution


## Function
    function add (a, b) {
        return a + b
    }
