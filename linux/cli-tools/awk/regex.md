# Regular Expression
    ^       => starts of line
    $       => ends of line

    .       => single char
    [...]   => char set
    [^...]  => exclusive char set
    |       => or

    ?       => 0 or one occurence of prev char
    +       => 1 or more occurence of prev char
    *       => 0 or more occurence of prev char

    ()      => grouping e.g. '/(Appple|Orange)/'
