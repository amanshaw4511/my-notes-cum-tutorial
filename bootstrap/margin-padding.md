- property
    m  => margin
    p  => padding

- sides
    t  => top
    b  => bottom
    s  => start/left
    b  => end/right

- size
    0 => 0 px
    1 => 0.25 rem
    2 => 0.5  rem
    3 => 1    rem
    4 => 1.5  rem
    5 => 3    rem

# margin
<property><sides>-<size>
<property><sides>-<breakpoint>-<size>


# gutter
    gutters are the padding between the "col" of bootstrap
    it is used with "row" class

- horizontal gutter
    gx-<size>
- vertical gutter
    gy-<size>
- horizontal and vertical gutter
    g-<size>
- no gutter
    g-0
